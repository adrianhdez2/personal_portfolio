Este proyecto consistió en el **diseño y desarrollo web para AIRMEXTAB**, enfocado en construir una presencia digital moderna, funcional y alineada a sus necesidades.  

Se trabajó en una propuesta visual limpia y profesional, priorizando la claridad de la información y la experiencia del usuario.

**Objetivo:** Desarrollar una plataforma optimizada que permita comunicar sus servicios de forma clara, mejorar la interacción con los usuarios y facilitar el contacto.  

**Stack utilizado:**  
- Astro  
- React  
- Tailwind CSS  

**Características destacadas:**  
1. Interfaz moderna y minimalista enfocada en usabilidad.  
2. Arquitectura clara para presentar servicios e información relevante.  
3. Componentización con React para mayor escalabilidad.  
4. Optimización de rendimiento gracias a Astro.  
5. Diseño responsive adaptable a móviles y escritorio.  

📌 Todos los derechos de imágenes, logotipos y contenido visual pertenecen a sus respectivos autores y creadores.
